# Mission Control Dashboard

Central hub for monitoring OpenClaw agents, system stats, and budget tracking.

## Features

### 1. Budget Tracking (index.html)
- Daily/Weekly/Monthly AI usage costs
- Real-time updates every 5 minutes
- Cost optimization vs GPT-4 pricing
- Mobile-first responsive design

### 2. Agent Command Center (agents.html) ⭐ NEW
- **Real-time agent monitoring** - All active sessions and subagents
- **Token usage tracking** - Per-agent costs and burn rates
- **Context warnings** - Alerts when approaching token limits
- **Status indicators** - Active/Idle/Warning/Danger states
- **Auto-refresh** - Updates every 10 seconds

## Live URLs

- **Budget Dashboard:** https://mission-control-dashboard.pages.dev
- **Agent Center:** https://mission-control-dashboard.pages.dev/agents.html

## Quick Start

### Local Development

```bash
cd /root/.openclaw/workspace/projects/mission-control-dashboard

# Start web server
python3 -m http.server 8080

# Open in browser
open http://localhost:8080
open http://localhost:8080/agents.html
```

### Update Agent Data

The Agent Command Center requires `agents-data.json` to be generated periodically.

**Manual update:**
```bash
openclaw agent --local --message "Generate agent dashboard data"
```

**Automated (recommended):**
Set up a cron job to update every 10 seconds (see skills/agent-control-center/SKILL.md).

## Agent Command Center Details

### Metrics Tracked

- **Session info** - Key, display name, channel
- **Model** - Which AI model is running
- **Tokens** - Input/output/total token counts
- **Context usage** - Percentage of context window filled
- **Runtime** - How long the session has been active
- **Cost** - Estimated API cost based on model pricing
- **Burn rate** - Cost per hour (tokens/hour × model price)

### Status Indicators

- 🟢 **Active** - Updated in last 5 minutes
- 🟠 **Warning** - Context >80% OR burn rate >$2/hr
- 🔴 **Danger** - Context >90% OR burn rate >$5/hr
- ⚪ **Idle** - Not updated recently

### Cost Calculation

Real pricing for common models:
- Claude Sonnet 4.5: $3/M input, $15/M output
- Claude Opus 4: $15/M input, $75/M output  
- Gemini 2.5 Flash: $0.075/M input, $0.30/M output
- GPT-4: $30/M input, $60/M output

Assumes 70% input / 30% output ratio.
